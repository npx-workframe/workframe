## Source stance

This version incorporates the revised deployment and payment mode strategy:

1. Local self-host with Win/Mac/Hermes install.
2. Docker self-host for trusted teams.
3. VPS self-host at a customer domain with invite/domain allowlist.
4. Workframe-provisioned hosted or BYOC cells on VPS/server clusters.
5. Enterprise BYOC with SSO and support.

It also expands payment modes from BYOK to company-level provider keys, hosted credits, run-level billing chains, marketplace settlement, and future agent-to-agent payment rails.
