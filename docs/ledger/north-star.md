# North star (summary)

**Full brief:** [strategy/workframe_v0_1_1_docs/WORKFRAME_v0.1.1_MASTER.md](../strategy/workframe_v0_1_1_docs/WORKFRAME_v0.1.1_MASTER.md)  
**Tagline:** The Social OS for Autonomous Businesses

## First principle

```text
humans + persistent agents + computers = autonomous business cell
```

Runtimes (Hermes, Codex, Pi, …) and models are **replaceable**. Workframe owns identity, cells, agent registry, runs, policies, brokers, files, audit, billing.

## Stable vs replaceable

| Workframe owns | Replaceable adapters |
|----------------|---------------------|
| Workspaces / cells | Model providers |
| Persistent agents | LLM harnesses |
| Boards, files, rooms | CLI tools |
| Run ledger | Sandboxes |
| Credential / tool / egress brokers | Connectors |
| Billing, policies | Cloud hosts |

## Build next (authority layer)

```text
run ledger
credential leases everywhere
tool brokers
file manifests
egress policy (unavoidable broker)
runtime adapters
cell manager
billing line items
```

**Do not** over-invest in owning the agent brain. Keep Hermes strong; add adapters as the ecosystem evolves.

## Wedge vs vision

| Now (v0.1.x) | North star |
|--------------|------------|
| Hermes Docker runtime | Adapter-first cell |
| Lease + LLM proxy | Full credential/egress broker |
| Invite-only multi-user | Deployment mode matrix + hosted cells |
| Activity / chat / boards | Run ledger + marketplace |

Current shipping scope is intentional: prove install, security primitives, and team collaboration before cell manager and billing.
