# Workframe work ledger

`ledger.json` is the only current-work authority and implements
`architectonic-rail@0.3`. `north-star.md` and strategy docs express durable
intent; specs and audits provide depth and evidence; the harness proves runtime
behavior; Git and release notes preserve history.

Backlog, queue, and now are derived views, not parallel files. Agents select one
dependency-clear `WF-*` item, inspect its named source, preserve stop lines,
verify acceptance, and record evidence plus one lifecycle transition. A bounded
interactive change completed and verified in-session needs no ceremonial item.
