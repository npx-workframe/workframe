# Workframe PM composition

```text
runtime constitution + pm-workframe role + Workframe project binding +
docs/ledger/ledger.json item + named source/evidence
```

Specs in `docs/ledger/specs/<WF-ID>/` deepen one item; they do not create a
second task list. The harness supplies verification evidence. Architectonic Rail
owns the lifecycle contract, and ABKB routes to this source ledger without
mirroring it.
