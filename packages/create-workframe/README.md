# create-workframe

Published on npm as **create-workframe**.

```bash
npx create-workframe@0.1.2 MyProject
```

Scaffolds an isolated Workframe + Hermes project on Windows, macOS, and Linux.

## Generated layout

```text
MyProject/
├── Agents/
├── Files/
├── docker-compose.yml
├── workframe-api/
├── workframe-ui/
└── workframe-manifest.json
```
